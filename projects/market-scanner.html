ğŸ“Š Analysis Components

1. Market Regime Detection

The system continuously monitors overall market conditions:

  • VIX Analysis - Volatility index with multi-day momentum tracking
  • SPY vs SMA200 - Broad market trend assessment
  • RSP (Equal Weight) - Market participation breadth
  • Breadth Analysis - % of S&P 500 above 50-day SMA

2. ML Jury System

Multi-model ensemble voting for signal confidence:

Model Type Strengths
Random Forest Ensemble Trees Handles non-linear relationships, robust to outliers
XGBoost Gradient Boosting High accuracy, handles missing data
MLP Neural Network Captures complex patterns, learns representations

3. Signal Priority System

Priority Category Description
1 GOLDEN Multiple confirming indicators + strong ML consensus
2 SILVER Single strong indicator + ML consensus
3 TREND Moving average crossover signals
4 OTHER Remaining valid signals

4. Sell Signal Detection

Automated warning system for regime deterioration:

  • Breadth Deterioration - Declining market participation
  • A/D Collapse - Advance/Decline ratio breakdown
  • VIX Spike - Sudden volatility increase
  • Failed Thrust - Aborted breadth expansion
  • Divergence - Price vs breadth disagreement

Live Breadth Scan 30 minutes Full S&P 500 scan
ML Recommendations On upload Via sync script
